# Marketing Mix Modeling (MMM)

## Objective
Measure the effectiveness of marketing channels and estimate their contribution to sales.

## Methodology
- Adstock Transformation
- Saturation Curves
- OLS Regression
- Contribution Analysis
- ROI Analysis
- VIF Diagnostics

## Tools Used
- Python
- Pandas
- Statsmodels
- Matplotlib

## Results
- Built an end-to-end MMM framework
- Estimated channel contributions
- Calculated marketing ROI
- Validated model using Actual vs Predicted analysis
